# Iconfont 需求清单

本清单用于后续从 iconfont 下载统一图标资源。本轮只整理图标位点和建议命名，不接入图标文件。

## 侧边栏导航

| 图标 key | 使用位置 | 建议含义 |
| --- | --- | --- |
| `icon-dashboard` | 天下纵览 / 世界概览 | 仪表盘、地球、总览 |
| `icon-summit` | 紫禁之巅 | 山峰、奖杯、阵容 |
| `icon-nation` | 国家圣殿、国家概览、国家荣誉 | 旗帜、盾牌 |
| `icon-club` | 豪门殿堂、豪门概览、豪门荣誉 | 球场、徽章、建筑 |
| `icon-star` | 巨星殿堂、巨星概览、巨星荣誉 | 星星、球员 |
| `icon-tianji` | 天机阁 | 设置、齿轮、控制台 |
| `icon-rule` | 荣誉规则 | 规则、清单、权重 |
| `icon-competition` | 赛事管理 | 赛程、奖杯、对阵 |
| `icon-award` | 奖项管理 | 奖牌、桂冠 |
| `icon-import` | 数据导入 | 上传、导入 |
| `icon-config` | 基础配置 | 调节器、配置 |
| `icon-remark` | 备注管理 | 文档、批注 |
| `icon-audit` | 权限与审计 | 盾牌、日志 |

## 通用操作

| 图标 key | 使用位置 | 建议含义 |
| --- | --- | --- |
| `icon-search` | 全局搜索、列表筛选 | 放大镜 |
| `icon-add` | 新增按钮 | 加号 |
| `icon-edit` | 编辑按钮 | 铅笔 |
| `icon-delete` | 删除按钮 | 垃圾桶 |
| `icon-save` | 保存按钮 | 保存、对勾 |
| `icon-cancel` | 取消按钮 | 关闭、返回 |
| `icon-refresh` | 刷新、重算 | 刷新、循环箭头 |
| `icon-filter` | 筛选 | 漏斗 |
| `icon-reset` | 重置 | 回退、清空 |
| `icon-back` | 返回列表 | 左箭头 |
| `icon-link` | 外部链接 | 链接、跳转 |
| `icon-close` | 关闭 tab / 弹窗 | 关闭 |
| `icon-close-others` | 关闭其他 tab | 多窗口关闭 |
| `icon-more` | 更多操作 | 更多、省略号 |

## 数据对象

| 图标 key | 使用位置 | 建议含义 |
| --- | --- | --- |
| `icon-confederation` | 足联选择器、基础配置 | 联盟、网络 |
| `icon-country` | 国家选择器、国家详情 | 旗帜 |
| `icon-club-badge` | 俱乐部选择器、俱乐部详情 | 队徽 |

| `icon-position` | 位置配置、位置筛选 | 球场位置 |
| `icon-trophy` | 冠军、奖杯数 | 奖杯 |
| `icon-medal` | 奖牌、名次 | 奖牌 |
| `icon-score` | 荣誉分、评分 | 分数、趋势 |
| `icon-user` | Header 账户头像 | 用户 |
| `icon-language` | Header 语言切换 | 语言、翻译 |
