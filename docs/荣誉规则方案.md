# 荣誉规则方案

本文档记录当前系统默认的赛事荣誉分规则。规则按结构字段命中赛事，不按世界杯、欧冠等赛事名称硬判断；“典型命中赛事”只用于后台展示和人工理解，不参与普通规则匹配。

## 计算公式

单条名次荣誉分：

```text
名次分 × 质量系数 × 自动换算系数
```

- 名次分：规则行中的冠军分、亚军分、季军分、殿军分。
- 质量系数：足联系数或国家系数；没有命中特定系数时使用规则默认质量系数。
- 自动换算系数：只在指定换算方式启用，默认不换算。

默认名次系数用于生成系统初始分值：

| 名次 | 默认系数 |
| --- | ---: |
| 冠军 | 1 |
| 亚军 | 0.5 |
| 季军 | 0.3 |
| 殿军 | 0.2 |

国内联赛采用“联赛前三”：冠军 1、亚军 0.4、季军 0.25，不计殿军。

## 名次范围

| 规则值 | 展示 | 说明 |
| --- | --- | --- |
| `TOP_FOUR` | 冠亚季殿 | 冠军、亚军、季军、殿军都计分 |
| `TOP_THREE` | 前三 | 冠军、亚军、季军计分 |
| `TOP_TWO` | 冠亚 | 冠军、亚军计分 |
| `LEAGUE_TOP_THREE` | 联赛前三 | 国内联赛冠军、亚军、季军计分 |
| `CHAMPION_ONLY` | 仅冠军 | 只计冠军 |

当前口径：

- 国家队国际和洲际赛事使用 `TOP_FOUR`。
- 俱乐部国际一级杯赛使用 `TOP_FOUR`。
- 俱乐部洲际赛、国内杯赛、其他杯赛默认使用 `TOP_TWO`。
- 国内联赛使用 `LEAGUE_TOP_THREE`，不计殿军。

## 自动换算

| 规则值 | 使用场景 | 说明 |
| --- | --- | --- |
| `NONE` | 默认 | 不做额外换算 |
| `FREQUENCY_SCALE` | 国家队洲际一级杯赛 | 根据届次间隔和参赛数量自动换算 |
| `OLYMPIC_STAGE` | 国家队国际三级杯赛 | 奥运会按历史阶段换算 |
| `CLUB_WORLD_CUP_STAGE` | 俱乐部国际一/二级杯赛 | 预留给世俱杯阶段换算 |

### 频率系数

后台根据赛事已录入届次年份计算平均间隔：

| 平均间隔 | 系数 |
| --- | ---: |
| 大于等于 3.5 年 | 1 |
| 大于等于 2.5 年 | 0.85 |
| 大于等于 1.5 年 | 0.7 |
| 小于 1.5 年 | 0.45 |

### 规模系数

后台优先使用当前届次数量；当前届次没有数量时，使用该赛事所有已录入届次数量的中位数：

| 参赛数量 | 系数 |
| --- | ---: |
| 大于等于 24 | 1 |
| 大于等于 16 | 0.9 |
| 大于等于 10 | 0.75 |
| 大于等于 8 | 0.65 |
| 大于等于 4 | 0.5 |
| 未录入或低于 4 | 1 |

### 奥运阶段换算

| 年份 | 系数 |
| --- | ---: |
| 1928 年及以前 | 2.5 |
| 1929-1988 年 | 1.2 |
| 1989 年及以后 | 0.75 |

### 世俱杯阶段换算

当前规则结构已预留 `CLUB_WORLD_CUP_STAGE`，第一版不额外改变分值；后续可按世俱杯赛制阶段补充换算。

## 命中优先级

后台按以下结构字段匹配：

1. 对象类型。
2. 分类。
3. 级别。
4. 赛制。
5. 适用范围。
6. 足联或国家范围。

如果多条规则同时命中，优先使用足联、国家、范围、赛制、级别、分类信息更完整的规则。典型命中赛事不参与匹配。
