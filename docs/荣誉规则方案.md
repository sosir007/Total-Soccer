# 荣誉规则方案

本文档记录当前系统默认的赛事荣誉分规则。规则按结构字段命中赛事，不按世界杯、欧冠等赛事名称硬判断；“典型命中赛事”只用于后台展示和人工理解，不参与普通规则匹配。

## 计算公式

单条名次荣誉分：

```text
名次分 × 质量系数 × 自动换算系数
```

- 名次分：规则行中的冠军分、亚军分、季军分、殿军分、四强分。
- 质量系数：足联系数或国家系数；没有命中特定系数时使用规则默认质量系数。
- 自动换算系数：只在指定换算方式启用，默认不换算。

默认名次系数用于生成系统初始分值：

| 名次 | 默认系数 |
| --- | ---: |
| 冠军 | 1 |
| 亚军 | 0.5 |
| 季军 | 0.3 |
| 殿军 | 0.2 |
| 四强 | 季军和殿军平均 |

国内联赛采用“联赛前三”：冠军 1、亚军 0.4、季军 0.25，不计殿军。

## 届次名次口径

“是否有三四名赛”不是赛事固定属性，而是每一届的录入属性。比如欧洲杯早期有三四名赛，1984 年起没有三四名赛；同一项赛事需要按届次区分。

| 规则值 | 展示 | 允许录入 |
| --- | --- | --- |
| `THIRD_PLACE_MATCH` | 有三四名赛 | 冠军、亚军、季军、殿军 |
| `SEMI_FINALISTS` | 无三四名赛 | 冠军、亚军、四强 1、四强 2 |
| `FINAL_ONLY` | 仅冠亚军 | 冠军、亚军 |
| `LEAGUE_TOP_THREE` | 联赛前三 | 冠军、亚军、季军 |
| `DOUBLE_THIRD_PLACE` | 并列季军 | 冠军、亚军、季军 1、季军 2 |

录入原则：

- 有真实三四名赛的届次，录入季军和殿军。
- 没有三四名赛但存在半决赛的届次，录入两个四强，不人为指定季军和殿军。
- 三四名赛并列铜牌的届次，录入两个季军，不录殿军。
- 只统计决赛结果的杯赛，使用仅冠亚军。
- 国内联赛使用联赛前三，不计殿军和四强。

## 名次范围

| 规则值 | 展示 | 说明 |
| --- | --- | --- |
| `TOP_FOUR` | 冠亚季殿/四强 | 有三四名赛时计冠军、亚军、季军、殿军；无三四名赛时计冠军、亚军、两个四强 |
| `TOP_THREE` | 前三 | 冠军、亚军、季军计分 |
| `TOP_TWO` | 冠亚 | 冠军、亚军计分 |
| `LEAGUE_TOP_THREE` | 联赛前三 | 国内联赛冠军、亚军、季军计分 |
| `CHAMPION_ONLY` | 仅冠军 | 只计冠军 |

当前口径：

- 国家队国际和洲际赛事使用 `TOP_FOUR`。
- 俱乐部国际一级杯赛使用 `TOP_FOUR`。
- 俱乐部洲际赛、国内杯赛、其他杯赛默认使用 `TOP_TWO`。
- 国内联赛使用 `LEAGUE_TOP_THREE`，不计殿军。

统计口径：

- 奖牌数只包含冠军、亚军、季军。
- 总荣誉数包含冠军、亚军、季军、殿军、四强。
- 四强分只在 `TOP_FOUR` 规则中启用，默认取季军分和殿军分的平均值。

## 自动换算

| 规则值 | 使用场景 | 说明 |
| --- | --- | --- |
| `NONE` | 默认 | 不做额外换算 |
| `FREQUENCY_SCALE` | 国家队洲际一级杯赛 | 根据届次间隔和参赛数量自动换算 |
| `OLYMPIC_STAGE` | 国家队国际三级杯赛 | 奥运会按历史阶段换算 |
| `CLUB_WORLD_CUP_STAGE` | 俱乐部国际一级杯赛 | 世俱杯按历史阶段换算 |

### 频率系数

后台根据赛事已录入届次记录计算平均间隔；同一年多届按多个荣誉机会处理，不按年份去重：

```text
平均间隔 = (最后一届年份 - 第一届年份) / (届次数 - 1)
频率系数 = min(平均间隔 / 4, 1)
```

| 平均间隔 | 系数示例 |
| --- | ---: |
| 4 年及以上 | 1 |
| 3 年 | 0.75 |
| 2 年 | 0.5 |
| 1 年 | 0.25 |

### 规模系数

后台优先使用当前届次数量；当前届次没有数量时，使用该赛事所有已录入届次数量的中位数：

| 参赛数量 | 系数 |
| --- | ---: |
| 大于等于 24 | 1 |
| 大于等于 16 | 0.9 |
| 大于等于 10 | 0.75 |
| 大于等于 8 | 0.65 |
| 大于等于 4 | 0.5 |
| 3 | 0.35 |
| 2 | 0.25 |
| 未录入 | 1 |
| 其他异常数量 | 0 |

### 奥运阶段换算

| 年份 | 系数 |
| --- | ---: |
| 1908-1928 年 | 3 |
| 1936-1980 年 | 2 |
| 1984-1988 年 | 1.5 |
| 1992 年及以后 | 1 |

当前国家队国际三级杯赛规则基础冠军分为 4，因此奥运男足冠军分按阶段分别为 12、8、6、4。奥运会不叠加频率/规模换算；泛美锦标赛等普通国家队国际三级赛事按 4 分基础分计算。

### 世俱杯阶段换算

世俱杯只保留一个赛事，统一命中俱乐部国际一级杯赛规则，并按年份阶段换算：

| 年份范围 | 系数 | 冠军分 |
| --- | ---: | ---: |
| 2025 年以前 | 0.5 | 8 |
| 2025 年及以后 | 1 | 16 |

俱乐部其他全球赛事不走世俱杯阶段换算：

- 2024 年起的新国际足联洲际杯命中 `CLUB_INTERNATIONAL_LEVEL_2_CUP`，冠军分 6。
- 1960-2004 年的丰田杯 / 旧洲际杯命中 `CLUB_INTERNATIONAL_LEVEL_3_CUP`，冠军分 4。
- 1968-1969 年的洲际冠军超级杯命中 `CLUB_INTERNATIONAL_LEVEL_4_CUP`，冠军分 2。

## 团队附加分

团队附加分用于补充国家队和俱乐部的荣誉分，只记录白名单内的官方团队奖、权威年度团队奖和少量历史稳定媒体奖。附加分不替代赛事成绩，也不改变赛事 standings 的冠军、亚军、季军等名次分。

通用原则：

- 附加分只作为小额补充，不能高于同层级核心赛事成绩。
- 官方赛事内团队奖优先于年度机构评选，年度机构评选优先于媒体或商业奖项。
- 公平竞赛奖必须能确认是赛事官方奖，普通精神文明奖、球迷票选奖、赞助商奖不计分。
- 月度榜、Top 10 榜单、世纪最佳、十年最佳、历史排名等非常规荣誉不进入荣誉分；如有展示价值，只作为备注荣誉。
- 遇到新奖项时先进入待确认清单，确认权威性、获奖对象和计分档位后再录入。

### 国家队附加分

| 奖项类型 | 典型奖项 | 分值 | 录入边界 |
| --- | --- | ---: | --- |
| 官方公平竞赛奖 | 世界杯公平竞赛奖、洲际杯官方公平竞赛奖、FIFA Fair Play Award | 1 | 获奖对象必须能明确对应国家队或足协代表队 |
| 权威年度团队奖 | 劳伦斯年度最佳团队，获奖对象为国家队 | 2 | 只录最终获奖者，不录提名 |
| 年度最佳国家队 | IFFHS 世界最佳国家队 / 年度国家队排名第一 | 1 | 只录年度最终结果，不录月度榜或阶段榜 |
| 稳定媒体团队奖 | World Soccer Team of the Year 选中国家队 | 1 | 只录年度最终获奖对象 |

暂不计分：

- 普通球迷票选、商业活动奖、纪念奖。
- 世纪最佳国家队、十年最佳国家队、历史国家队排名。
- 无法确认获奖对象是国家队还是足协组织的奖项。

| 名次 | 系数 | 分值 |
| --- | ---: | ---: |
| 第一名 | 1 | 2 |
| 第二名 | 0.5 | 1 |

录入边界：

- IFFHS 只录年度最终榜前三；如果资料只确认第一名，不强补第二、第三名。
- 如果官方出现并列名次，按官方排名记录；分值仍按对应名次系数计算。
- 世纪最佳俱乐部、十年最佳俱乐部、历史最佳俱乐部、长期排名榜不进入荣誉分。
- Globe Soccer 等商业属性较强的奖项必须白名单确认，不因同类奖项自动扩展。

### 落库前说明

当前系统主要支持赛事 standings 和球员个人奖项。国家队 / 俱乐部团队奖项后续应单独设计团队奖项对象或团队附加荣誉模型，再接入荣誉分重算；不要把团队奖硬塞进球员个人奖项表。

## 命中优先级

后台按以下结构字段匹配：

1. 对象类型。
2. 分类。
3. 级别。
4. 赛制。
5. 适用范围。
6. 足联或国家范围。

如果多条规则同时命中，优先使用足联、国家、范围、赛制、级别、分类信息更完整的规则。典型命中赛事不参与匹配。
