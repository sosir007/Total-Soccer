# 俱乐部缺失清单

用于集中记录“某项赛事补录时，因数据库暂未收录而留空的俱乐部”。后续不再按单个赛事拆分新文档，而是在本文件内按赛事持续追加。

## 字段说明

- `国家`：俱乐部所属国家。
- `中文名`：当前建议使用的中文名称，后续入库时可再按你最终口径微调。
- `英文名`：本轮补录数据里使用的英文写法或常见原名。
- `所属赛事`：本次缺失是在哪个赛事补录过程中发现的。
- `所属赛事名次`：只记录当时第一次被记入缺失清单时，对应那次补录里的具体赛事年份与名次；不追记后续其他赛事。
- `备注`：用于记录重名风险、译名待确认、后续需核对 UID 等信息。
