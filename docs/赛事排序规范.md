# 赛事排序规范

## 目标

赛事会越来越多，尤其是俱乐部赛事会覆盖国际、洲际、各国联赛和杯赛。为了避免所有赛事挤在一个全局排序数字里，赛事列表采用“分组优先，组内排序”的规则。

`sortOrder` 只表示同一分组内的展示顺序，不再承担全局排序职责。

## 列表排序规则

赛事列表按以下顺序排序：

1. 对象：国家队优先，俱乐部其次。
2. 范围：国际、洲际、国内、自定义/其他。
3. 分类：国际、洲际、国内、其他。
4. 级别：一级、二级、三级、四级、未设置。
5. 足联：按基础配置中的足联排序。
6. 国家：按国家 UID 排序。
7. 组内排序：按赛事自身 `sortOrder`。
8. 名称：中文名称兜底排序。

## 推荐分组

### 国家队国际赛事

用于世界杯、联合会杯、奥运男足等。

- `targetType`: `COUNTRY`
- `scopeType`: `GLOBAL`
- `category`: `国际`
- `level`: 按赛事重要性填写
- `sortOrder`: 只在国家队国际赛事内部排序

推荐：

| sortOrder | 赛事 |
| ---: | --- |
| 0 | 国际足联世界杯 |
| 10 | 国际足联联合会杯 |
| 15 | 奥运会男子足球赛 |

### 国家队洲际赛事

用于欧洲杯、美洲杯、亚洲杯、非洲杯、金杯、大洋洲杯等。

- `targetType`: `COUNTRY`
- `scopeType`: `CONFEDERATION`
- `category`: `洲际`
- `level`: 通常为 `一级`
- `scopeConfederations`: 填对应足联
- `sortOrder`: 只在同足联同级别赛事内部排序

如果同一足联未来有多项赛事，可以使用：

| sortOrder | 含义 |
| ---: | --- |
| 10 | 该足联最高级国家队杯赛 |
| 20 | 次级国家队杯赛 |
| 30 | 其他国家队赛事 |

### 俱乐部国际赛事

用于世俱杯、国际足联洲际杯、丰田杯/旧洲际杯等全球俱乐部赛事。

- `targetType`: `CLUB`
- `scopeType`: `GLOBAL`
- `category`: `国际`
- `level`: 按规则分级
- `sortOrder`: 只在俱乐部国际赛事内部排序

推荐：

| sortOrder | 赛事 |
| ---: | --- |
| 10 | 国际足联俱乐部世界杯 |
| 20 | 国际足联洲际杯 |
| 30 | 丰田杯 / 旧洲际杯 |

### 俱乐部洲际赛事

用于欧冠、欧联、南美解放者杯、亚冠等。

- `targetType`: `CLUB`
- `scopeType`: `CONFEDERATION`
- `category`: `洲际`
- `level`: 按赛事重要性填写
- `scopeConfederations`: 填对应足联
- `sortOrder`: 只在同足联同级别赛事内部排序

推荐：

| sortOrder | 含义 |
| ---: | --- |
| 10 | 顶级洲际俱乐部赛事，例如欧冠、解放者杯 |
| 20 | 二级洲际俱乐部赛事，例如欧联 |
| 30 | 三级或其他洲际俱乐部赛事 |

### 俱乐部国内赛事

用于英超、意甲、西甲、足总杯、国王杯、社区盾等。

- `targetType`: `CLUB`
- `scopeType`: `COUNTRY`
- `category`: `国内`
- `level`: 按赛事重要性填写
- `scopeCountries`: 填对应国家
- `sortOrder`: 只在同国家同级别赛事内部排序

推荐：

| sortOrder | 含义 |
| ---: | --- |
| 10 | 顶级联赛，例如英超、西甲、意甲 |
| 20 | 主要杯赛，例如足总杯、国王杯 |
| 30 | 次级杯赛，例如联赛杯 |
| 40 | 超级杯、社区盾等 |

## 注意事项

- 不要再为了全局位置给 `sortOrder` 留大段空档。
- 新增赛事时优先把 `targetType / scopeType / category / level / scopeConfederations / scopeCountries` 填准确。
- 如果赛事跨多个国家或足联，优先使用多范围字段；列表排序会取其中最靠前的足联或国家。
- 页面手动调整 `sortOrder` 时，只需要考虑它在当前分组里的顺序。
- 赛事荣誉规则匹配不依赖列表排序，排序只影响后台列表和下拉展示顺序。
