# 赛事排序规范

## 目标

赛事会越来越多，尤其是俱乐部赛事会覆盖国际、洲际、各国联赛和杯赛。为了避免所有赛事挤在一个全局排序数字里，赛事列表采用“分组优先，组内排序”的规则。

`sortOrder` 只表示同一分组内的展示顺序，不再承担全局排序职责。推荐优先从赛事相关的足联 UID 或国家 UID 派生排序值，方便后续补充赛事时保持稳定顺序。

## 列表排序规则

赛事列表按以下顺序排序：

1. 对象：国家队优先，俱乐部其次。
2. 范围：国际、洲际、国内、自定义/其他。
3. 分类：国际、洲际、国内、其他。
4. 级别：一级、二级、三级、四级、未设置。
5. 赛制：联赛、杯赛、其他。
6. 足联：按基础配置中的足联排序。
7. 国家：按国家 UID 排序。
8. 状态：现行优先，停办其次。
9. 组内排序：按赛事自身 `sortOrder`。
10. 名称：中文名称兜底排序。

## 推荐分组

### 国家队国际赛事

用于世界杯、联合会杯、奥运男足等。

- `targetType`: `COUNTRY`
- `scopeType`: `GLOBAL`
- `category`: `国际`
- `level`: 按赛事重要性填写
- `sortOrder`: 只在国家队国际赛事内部排序

推荐：

| sortOrder | 赛事 |
| ---: | --- |
| 0 | 国际足联世界杯 |
| 1 | 国际足联联合会杯 |
| 2 | 奥运会男子足球赛 |
| 3 | 泛美锦标赛 |

### 国家队洲际赛事

用于欧洲杯、美洲杯、亚洲杯、非洲杯、金杯、大洋洲杯等。

- `targetType`: `COUNTRY`
- `scopeType`: `CONFEDERATION`
- `category`: `洲际`
- `level`: 通常为 `一级`
- `scopeConfederations`: 填对应足联
- `sortOrder`: 只在同足联同级别赛事内部排序

洲际赛事按足联 UID 乘以 10 起步，同一足联内继续累加。

| sortOrder | 赛事 |
| ---: | --- |
| 10 | 非洲国家杯 |
| 20 | 亚足联亚洲杯 |
| 30 | 欧洲足球锦标赛 |
| 31 | 欧足联国家联赛等欧足联后续赛事 |
| 40 | 中北美及加勒比海金杯赛 |
| 50 | 大洋洲国家杯 |
| 60 | 南美足联美洲杯 |

跨足联、跨区域或没有单一足联归属的国家队赛事优先使用 `CUSTOM`，从 `70` 开始排序；一般其他国家队赛事从 `100` 开始排序。

### 俱乐部国际赛事

用于世俱杯、国际足联洲际杯、丰田杯/旧洲际杯等全球俱乐部赛事。

- `targetType`: `CLUB`
- `scopeType`: `GLOBAL`
- `category`: `国际`
- `level`: 按规则分级
- `sortOrder`: 只在俱乐部国际赛事内部排序

推荐：

| sortOrder | 赛事 |
| ---: | --- |
| 0 | 国际足联俱乐部世界杯 |
| 1 | 国际足联洲际杯 |
| 2 | 丰田杯 / 旧洲际杯 |
| 3 | 洲际冠军超级杯 |

### 俱乐部洲际赛事

用于欧冠、欧联、南美解放者杯、亚冠等。

- `targetType`: `CLUB`
- `scopeType`: `CONFEDERATION`
- `category`: `洲际`
- `level`: 按赛事重要性填写
- `scopeConfederations`: 填对应足联
- `sortOrder`: 只在同足联同级别赛事内部排序

洲际俱乐部赛事按足联 UID 乘以 10 起步，同一足联内继续累加。

| sortOrder | 含义 |
| ---: | --- |
| 30 | 欧足联顶级俱乐部赛事，例如欧冠 |
| 31 | 欧足联二级俱乐部赛事，例如欧联 |
| 32 | 欧足联三级俱乐部赛事，例如欧协联 |
| 60 | 南美足联顶级俱乐部赛事，例如解放者杯 |

其他足联俱乐部赛事同样按对应足联 UID 乘以 10 起步，例如亚足联俱乐部赛事从 `20` 开始。

### 俱乐部国内赛事

用于英超、意甲、西甲、足总杯、国王杯、社区盾等。

- `targetType`: `CLUB`
- `scopeType`: `COUNTRY`
- `category`: `国内`
- `level`: 按赛事重要性填写
- `scopeCountries`: 填对应国家
- `sortOrder`: 只在同国家、同级别、同赛制、同状态赛事内部排序

国内俱乐部赛事按国家 UID 乘以 10 起步，`sortOrder` 不再承担“一级联赛、二级联赛、一级杯赛”等固定槽位含义；赛事列表会先按 `level` 和 `format` 分组，再按状态和 `sortOrder` 排序。

同一国家下可以让多个不同级别或不同赛制的赛事使用同一个基础 `sortOrder`。例如美国 UID 为 `390`，美国国内赛事可从 `3900` 起步；如果 MLS 与旧北美足球联赛都填 `3900`，列表会先按现行/停办排序，现行赛事排在停办赛事前。

| sortOrder | 含义 |
| ---: | --- |
| 7650 | 英格兰国内一级联赛，例如英超 |
| 7650 | 英格兰国内二级联赛，例如英冠；靠 `level=二级` 排在一级联赛之后 |
| 7650 | 英格兰国内一级杯赛，例如足总杯；靠 `format=杯赛` 排在同级联赛之后 |
| 7960 | 西班牙国内一级联赛，例如西甲 |
| 7960 | 西班牙国内一级杯赛，例如国王杯 |
| 16510 | 巴西国内一级联赛，例如巴甲 |
| 16510 | 巴西国内二级联赛，例如巴乙；靠 `level=二级` 排在一级联赛之后 |
| 16510 | 巴西国内一级杯赛，例如巴西杯；靠 `format=杯赛` 排在同级联赛之后 |
| 16518 | 巴西州级联赛统一排序号，例如保利斯塔锦标赛、卡里奥卡锦标赛、米内罗锦标赛 |
| 16519 | 巴西州级 / 地区杯赛统一排序号，例如里约-圣保罗锦标赛、东北杯、绿色杯 |
| 3900 | 美国国内一级联赛，例如 MLS / 旧北美足球联赛；靠 `lifecycleStatus` 让现行赛事排在停办赛事前 |

如果某些国家 UID 与洲际赛事排序数字接近或重复，不需要额外避让；列表会先按对象和范围分组，再比较 `sortOrder`。一般其他俱乐部赛事从 `100` 开始排序。

#### 巴西州级 / 地区赛事排序

巴西比较特殊，州级联赛和跨州地区杯赛数量多，且在早期巴西足球荣誉资料中经常出现。为了避免 `16510` 段被大量州级赛事挤满，巴西州级和地区赛事使用固定排序号：

| sortOrder | 适用赛事 | 系统归类 | 说明 |
| ---: | --- | --- | --- |
| 16518 | 巴西州级联赛 | 俱乐部国内二级联赛 | 例如保利斯塔锦标赛、卡里奥卡锦标赛、米内罗锦标赛、高乔锦标赛 |
| 16519 | 巴西州级 / 地区杯赛 | 俱乐部国内三级杯赛 | 例如里约-圣保罗锦标赛、东北杯、绿色杯等，需逐项确认是否计分 |

注意：

- `16518 / 16519` 是巴西历史赛事的特殊排序口径，不代表新增独立荣誉规则。
- 赛事规则仍按 `targetType=CLUB`、`scopeType=COUNTRY`、`category=国内`、对应 `level / format` 匹配。
- 保利斯塔锦标赛等州级联赛先归入“俱乐部国内二级联赛”；里约-圣保罗锦标赛等重要地区杯赛先归入“俱乐部国内三级杯赛”。
- 普通州杯、短期邀请赛、季前赛不因使用 `16519` 自动计分，仍需单独判断是否只作备注荣誉。
- 同一排序号下不再细分每个州或地区，列表最后用赛事名称兜底排序。

## 注意事项

- 不要再为了全局位置给 `sortOrder` 留大段空档；优先使用相关足联 UID 或国家 UID 派生的排序段。
- 新增赛事时优先把 `targetType / scopeType / category / level / scopeConfederations / scopeCountries` 填准确。
- 如果赛事跨多个国家或足联，优先使用多范围字段；列表排序会取其中最靠前的足联或国家。
- 页面手动调整 `sortOrder` 时，国内赛事优先按“国家 UID × 10”填写；只有同国家、同级别、同赛制、同状态下还需要细分时，才继续在该段内微调。
- 赛事荣誉规则匹配不依赖列表排序，排序只影响后台列表和下拉展示顺序。
