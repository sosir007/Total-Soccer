# Honor Icons

这组 SVG 资源用于冠军、亚军、季军、殿军和四强展示，尺寸统一为 `256x256`，风格贴合项目现有的草坪绿、奖杯金、浅色玻璃感背景。

| 文件               | 含义 | 设计语义                       |
| ------------------ | ---- | ------------------------------ |
| `champion.svg`     | 冠军 | 金杯、盾牌、足球、星标         |
| `runner-up.svg`    | 亚军 | 银色奖章、盾牌、足球           |
| `third-place.svg`  | 季军 | 铜色奖杯、盾牌、足球           |
| `fourth-place.svg` | 殿军 | 绿色名次盾、金色绶带、足球     |
| `top-four.svg`     | 四强 | 四队晋级结构、球场线、四色席位 |

建议在页面中以 `48px`、`64px`、`96px` 三档使用；详情页或空状态可以使用 `128px` 以上尺寸。
